收集的一些实用工具

Everything

Everything是(voidtools)（中文主页）开发的一款文件搜索工具，官网描述为“基于名称实时定位文件和目录（Locate files and folders by name instantly）”。它体积小巧，界面简洁易用，快速建立索引，快速搜索，同时占用极低的系统资源，实时跟踪文件变化，并且还可以通过http或ftp形式分享搜索。查找文件的神器

Wox

官网:http://www.wox.one/
文档:http://doc.wox.one/zh/

Wox是一款快速启动工具
官网介绍:Wox是一款启动器。用于快速搜索本机安装的各种程序，文件。也可以通过插件快速搜索网络内容，比如今天天气如何，某某电影的评分是多少等等。

Wox是开源的，开源地址：http://www.github.com/qianlifeng/wox

全局快捷键Alt+Space

部分功能演示:

  • 结合Everything进行文件搜索
  • 快速查找并启动程序

  • 使用有道翻译插件进行翻译
  • 进行百度搜索或者谷歌搜索(前提是你有梯子)

    Listary

    官网:https://www.listary.com/

全局快捷键:两次Ctrl

既然说到了everything和Wox,那就不得不说这两个的合体listary

部分功能演示:

  • 文件搜索
  • 快速查找并启动程序

  • 进行百度搜索或者谷歌搜索(前提是你有梯子)
